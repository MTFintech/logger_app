## Description

An app to log our logs in Grafana.

## Getting started
Past the lib in your folders

Import the o LoggerApp in pubspec.yalm
example
```yaml
  logger_app:
    path: ./logger_app
```


## Usage


Start the LoggerApp using a user userId and key geted grafana
example
```dart
    LoggerApp(userId: '918535', key: 'glc_eyJvI..');
```
